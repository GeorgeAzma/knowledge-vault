### Vector Axioms

| Axiom                                                             | Statement         |
| ----------------------------------------------------------------- | ----------------- |
| Addition [[Associativity]]                                        | $u+(v+w)=(u+v)+w$ |
| Addition [[Commutativity]]                                        | $u+v=v+u$         |
| Addition [[Identity Element]]                                     | $v+0=v$           |
| Scalar Multiplication Compatibility with [[Field]] Multiplication | $a(bv)=(ab)v$     |
| Scalar Multiplication [[Identity Element]]                        | $1v=v$            |
| Scalar Multiplication [[Distributivity]] w.r.t. Addition          | $a(u+v)=au+av$    |
| Scalar Multiplication [[Distributivity]] w.r.t [[Field]] Addition | $(a+b)v=av+bv$    |
