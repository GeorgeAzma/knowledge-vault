**Oxidation number** represents **number of electrons gained/lost/shared** when forming [[bond]][[s]]

**+1 = lost electron, -1 = gained electron** in `H2O` **H** loses electron `+1` and **O** gains electron `-1`
because oxygen is more [[Electronegativity|electronegative]]

- **Free elements** pure form elements have no oxidation `O2, H2, Fe`
- **Monatomic ions** oxidation = ion charge `Na⁺ → +1, Cl⁻ → −1`
- **Neutral molecule** oxidation sum = 0
- **Polyatomic ion** oxidation sum = the ion’s charge

| Element | Common Oxidation   | Notes                              |
| ------- | ------------------ | ---------------------------------- |
| H       | +1, −1             | +1 with nonmetals, −1 with metals  |
| O       | −2, −1, +2         | −1 in [[Peroxide]][[s]], +2 in OF₂ |
| F       | −1                 | Always in compounds                |
| Cl      | −1, +1, +3, +5, +7 | Often −1 in salts                  |
| Br      | −1, +1, +3, +5, +7 | Similar to Cl                      |
| I       | −1, +1, +3, +5, +7 | Similar to Cl                      |
| Na      | +1                 | Alkali metal                       |
| K       | +1                 | Alkali metal                       |
| Li      | +1                 | Alkali metal                       |
| Mg      | +2                 | Alkaline earth metal               |
| Ca      | +2                 | Alkaline earth metal               |
| Al      | +3                 | Group 13 metal                     |
| Zn      | +2                 | Usually stable +2                  |
| Fe      | +2, +3             | Common transition metal states     |
| Cu      | +1, +2             | Common transition metal states     |
| Ag      | +1                 | Stable +1                          |
| Au      | +1, +3             | Transition metal                   |
| S       | −2, +4, +6         | Depends on compound                |
| N       | −3, +3, +5         | Depends on compound                |
| P       | −3, +3, +5         | Depends on compound                |
| C       | −4, +2, +4         | Depends on compound                |
> [!example] Oxidation
> - **H₂O** → H+1, O−2 `sum = 0`
> - **CO₃²⁻** → C+4, O−2 each `sum = −2`
> - **NaCl** → Na+1, Cl−1
