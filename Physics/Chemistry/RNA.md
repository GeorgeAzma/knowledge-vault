Same as [[DNA]] but
- `T => U (Uracil)` 
- has 1 strand
- G can weakly pair with U `wobble pair`
- RNA folds onto itself and pairs that way
### RNA Types
| Type          | Pairing Used For                                         |
| ------------- | -------------------------------------------------------- |
| **tRNA**      | Fold into a cloverleaf shape to carry amino acids        |
| **rRNA**      | Form ribosome's structural core                          |
| **mRNA**      | Has structured regions that regulate translation         |
| **ribozymes** | Fold into [[enzyme]]-like shapes that catalyze reactions |
