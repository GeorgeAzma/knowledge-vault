# Structure
Human DNA is **~2 meters long**, crammed into a nucleus **~6 micrometers** across
### Nucleotide
##### 1. Phosphate Group $\text{PO}_4$
- negatively charged
- links nucleotides together
##### 2. Deoxyribose Sugar
- 5-carbon ring
- missing 1 oxygen at 2' carbon
- carbons are numbered 1' 3' 5'
    - **1вҖІ carbon** where the base attaches
    - **3вҖІ carbon** has a free OH group, where the next nucleotide joins
    - **5вҖІ carbon** where the phosphate attaches
- phosphate on **5вҖІ** bonds to **OH on 3вҖІ** of previous nucleotide
##### 3. Nitrogenous Base

|   Name   | Letter | Pairs with |   Family   |
| :------: | :----: | :--------: | :--------: |
| Adenine  |   A    |     T      |   Purine   |
| Thymine  |   T    |     A      | Pyrimidine |
| Guanine  |   G    |     C      |   Purine   |
| Cytosine |   C    |     G      | Pyrimidine |

- **Purines** `double-ring structure, larger`
- **Pyrimidines** (single-ring structure, smaller):

- **Link Strength** 
    - `A-T` connected by 2 hydrogen bonds `weaker`
    - `G-C` connected by 3 hydrogen bonds `stronger`
### 2 Nucleotide Strands
they run in opposite directions `antiparallel`
- `5вҖІ вҶ’ вҶ’ вҶ’ вҶ’ вҶ’ вҶ’ 3вҖІ`
- `3вҖІ вҶҗ вҶҗ вҶҗ вҶҗ вҶҗ вҶҗ 5вҖІ
- strands are read 5' to 3'
### Double Helix
| Feature             | Measurement     |
| ------------------- | --------------- |
| Helix width         | 2 nm            |
| Rise per base pair  | 0.34 nm         |
| Full turn every     | 10.5 base pairs |
| Pitch `turn length` | 3.4 nm          |
Twisting create 2 grooves, these grooves are how the cell reads DNA without unzipping it
- **Major Groove** wider and deeper, proteins that read DNA `transcription factors, enzymes` primarily bind here because the bases are more exposed and readable
- **Minor Groove** narrower, some proteins and drugs bind here too
### DNA Forms
| Form      | Description                              | When                                   |
| --------- | ---------------------------------------- | -------------------------------------- |
| **B-DNA** | Classic right-handed helix, 10.5 bp/turn | Normal cellular conditions             |
| **A-DNA** | Wider, shorter, right-handed, 11 bp/turn | Dehydrated conditions; RNA-DNA hybrids |
| **Z-DNA** | Left-handed helix, zigzag backbone       | High salt; may play regulatory roles   |
DNA can adopt different conformations depending on conditions `B-DNA is the default
# DNA Encoding
- each strand has **letters** called nucleotide bases `4 total letters = 2 bits / letter` 
- strand letters are paired using pairing rules, so they contain same info `ATGC => TACG`
- DNA is read using 3-letter words called **codons**
- 4 letters / 3-letter words = $4^3$ = 64 total codons `6 bits / codon`
- there are **20 [[amino acids]]** that proteins can be made from
- codon encodes which amino acid to use for a protein
    - codons can encode `64` items, but there are only `20` amino acids = redundancy
    - `44`redundant codons point to same amino acid
