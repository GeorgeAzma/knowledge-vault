### Logic Gates
- `AND, OR, XOR`
- `NOT, NAND, NOR, XNOR`
- `IMPLIES` if `A` then `B` (`!A | B`)
- `NIMPLIES` if `A` then `!B` (`A & !B`)
- `MAJ` `1` if majority inputs are `1`
### Components
- `MUX` multiplexer, selects inputs via input index
- `DMUX` demultiplexer, sets output at input index to `1`
