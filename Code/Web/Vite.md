Fast hot reloading
### Quick HTML Page
``` bash
npm create vite@latest <project-name>
cd <project-name>
npm i
touch index.html
npm run dev

# index.html
<script type="module" src="/src/main.ts"></script>

# Delete useless stuff in src/ and App.ts
```
### React Template
``` bash
npm create vite@latest <project-folder> -- --template react-ts
```
