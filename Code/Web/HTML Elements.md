### Head / Metadata
``` html
<!DOCTYPE html>  declaration that defines the document type and HTML version
<html> html root
<head> metadata, links, scripts, and title
<title> tab title
<link> link to external resources like stylesheets
<style> internal CSS styles
<script> JavaScript code or external script reference
```
### Layout / Structure
``` html
<body> main content of the HTML document
<header> introductory content or navigation
<nav> section with navigation links
<main> main content unique to the document
<section> thematic grouping of content
<article> self-contained content like a blog post
<aside> sidebar content, related but separate from main
<footer> footer content, author info, copyright
```
### Text
``` html
<h1>..<h6> headings
<p> paragraph
<br> line break
<hr> divider (horizontal rule)
<b> bold
<i> italic
<mark> highlighted text
<strong> strong importance (usually bold)
<em> emphasized text (usually italic)
<small> smaller text
<code> inline code snippet
<pre> preserves new-lines and spaces (preformatted text)
<span> generic inline container
<div> generic block container
```
### Links / Media
``` html
<a> hyperlink to other pages or resources
<img> image
<figure> container for images with optional <figcaption>
<figcaption> caption for a <figure>
<iframe> inline frame for another page
<video> video media
<audio> audio media
<source> media source for <video> or <audio>
<canvas> drawable area for graphics/animations
<svg> scalable vector graphics
```
### Lists
``` html
<ul> unordered list
<ol> ordered list
<li> list item
<dl> description list
<dt> term in a description list
<dd> description of a term
```
### Table
``` html
<table> table container
<tr> table row
<th> table header cell
<td> table data cell
```
### Forms
``` html
<form> submittable data collection
<input> interactive form input
<textarea> multi-line text input
<select> dropdown selection
<option> option inside <select>
<label> label for form input
<button> clickable button
<fieldset> group related form controls
<legend> title for <fieldset>
```
### Misc
``` html
<details> collapsible content
<summary> title for <details>
```
