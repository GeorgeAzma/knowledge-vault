### Piano Roll
##### Shortcuts
- `ALT + A` arpeggiator `play notes sequentually instead of together`
- `ALT + Q` quantize `snap to grid`
- `ALT + L` quick legato `extend notes until next note`
- `ALT + R` randomizer
- `ALT + X` scale note parameters
- `ALT + O` LFO `make note params oscillate`

- `ALT + SHIFT + C` switch to cut tool `just C if not using keyboard as MIDI`

- `CTRL + U` slice notes to grid divisions
- `CTRL + B` duplicate notes
- `CTRL + L` legato `extend notes until they touch`

- `CTRL + SHIFT + ->` extend notes 

- `SHIFT + D` shorten notes to minimum step
